Introduction

Carsen is a car Rental Service management system that enables a person to manage multibased car rental business.

App functionalities:

  • Bases CRUD operation for Admin/View For Agent
  • Mechanics/Repair/Agents CRUD operations for Admin only
  • Cars/Clients/Orders CRUD for Agents and Admins

App Layers:

  1. Front-End: Angular Application
  2. Back-End: RESTful API made using NodeJs and Express
    Functionnalities included:
    - Jwt authentication/authorisation system
    - 100% covered by automated unit/integration tests
    - Developed Using TDD
  3. Data Base: MongoDB
